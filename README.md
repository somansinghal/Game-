# 🎮 Python Game Collection

A **collection of 6 classic mini-games built using Python** that run directly in the terminal.

This project was created to practice **Python programming, game logic, and user interaction** while building fun playable games.

---

# 🎮 Games Included

### 1️⃣ Tic-Tac-Toe

Classic 3x3 board game where two players try to align three marks in a row.

### 2️⃣ Rock-Paper-Scissors

Play against the computer in the famous decision game.

### 3️⃣ Guess The Number

Try to guess the number chosen by the computer with hints.

### 4️⃣ Hand Cricket (User vs Computer)

A fun cricket simulation game played with numbers.

### 5️⃣ Bingo (User vs User)

Two players compete to complete bingo rows first.

### 6️⃣ Memory Game (User vs User)

A memory challenge game where players recall sequences.

---

## ⚡ How to Run

```bash
git clone https://github.com/somansinghal/Game-.git
cd Game-
python main.py
```
